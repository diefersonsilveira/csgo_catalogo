This Android application provides a list with Counter-Strike: Global Offensive content (skins, crates, stickers and agents) using a public CS:GO API.

<br>

##  üîç Overview
<img src="Screenshots/csgo.gif" width="25%" align="right"/>

- Minimum SDK level 30  
- Target SDK level 36  

- Based on [Kotlin](https://kotlinlang.org/) + [Coroutines](https://github.com/Kotlin/kotlinx.coroutines)  

- UI built with XML layouts, [ViewBinding](https://developer.android.com/topic/libraries/view-binding), [RecyclerView](https://developer.android.com/guide/topics/ui/layout/recyclerview), and [ViewPager2](https://developer.android.com/guide/navigation/navigation-swipe-view-2)  

- Uses [Retrofit](https://square.github.io/retrofit/), [OkHttp Logging Interceptor](https://square.github.io/okhttp/features/interceptors/), [Glide](https://github.com/bumptech/glide), and [Coil](https://coil-kt.github.io/coil/) for networking and image loading
